# User Stories

## As an Admin
- **Story 1:** As an Admin, I want to add a new doctor to the system by providing their name, email, phone number, and specialty, so that they can be scheduled for appointments.
- **Story 2:** As an Admin, I want to view a list of all doctors in the system, so that I can manage their information.
- **Story 3:** As an Admin, I want to see a daily appointment report for a specific doctor, so that I can monitor clinic activity.

## As a Doctor
- **Story 1:** As a Doctor, I want to log in to my portal using my email and password, so that I can access my schedule and patient information securely.
- **Story 2:** As a Doctor, I want to view my list of appointments for the current day, so that I can prepare for my patient consultations.
- **Story 3:** As a Doctor, I want to set my available time slots for a specific day, so that patients can book appointments with me.
- **Story 4:** As a Doctor, I want to create a digital prescription for a patient after a consultation, so that the patient has a record of their medication.

## As a Patient
- **Story 1:** As a Patient, I want to log in to my portal using my email and password, so that I can manage my appointments and view my records.
- **Story 2:** As a Patient, I want to search for doctors by their specialty, so that I can find the right specialist for my needs.
- **Story 3:** As a Patient, I want to view a doctor's available time slots, so that I can book an appointment that fits my schedule.
- **Story 4:** As a Patient, I want to book an appointment with a doctor, so that I can receive medical consultation.
- **Story 5:** As a Patient, I want to view my upcoming appointments, so that I don't miss them.
- **Story 6:** As a Patient, I want to view my prescription history, so that I can keep track of my medications.
